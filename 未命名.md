# 悅和醬園拼圖遊戲

一款基於網頁的互動拼圖解謎遊戲，玩家需要在60秒內完成3x2網格的拼圖挑戰，拼出悅和醬園的產品圖像。

## 遊戲介紹

這是一個具有計時功能的拼圖遊戲，玩家需要從下方的碎片池中拖動拼圖碎片到上方的對應位置，完成整個圖像的拼接。遊戲特色包括：

- 60秒倒計時挑戰
- 3x2網格的拼圖布局
- 即時反饋拼圖正確性
- 完成挑戰後的慶祝效果（撒花動畫、禮物盒展示）
- 響應式設計，適配不同設備（手機、平板、電腦）

## 遊戲規則

1. 遊戲開始後，計時器將從60秒開始倒計時
2. 從下方碎片池中拖動拼圖碎片到上方對應位置
3. 正確放置的碎片會被固定，並顯示確認動畫
4. 錯誤放置的碎片會在2秒後返回池內，並顯示搖晃提示
5. 在規定時間內完成所有拼圖即可獲勝，獲得精美禮品；超時則挑戰失敗
